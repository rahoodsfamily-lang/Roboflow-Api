üí° How Webhooks Work

Webhooks allow you to receive real-time notifications when fire or smoke is detected.

Setup Steps:

  1. Create a webhook endpoint on your server that accepts POST requests
  2. Add the webhook URL above
  3. Choose which events trigger the webhook (fire, smoke, or both)
  4. Your endpoint will receive JSON data whenever a detection occurs
